#Space BommBoom

###Install

    #Clone this repo
	$ git clone https://github.com/DevSpaceBE/space-boomboom.git
	
	#Go to app's directory
	$ cd space-boomboom
	
	#Install dependecies
	$ npm install

###Run

    #Run server with webpack-dev/hot-middleware
    $ npm start

###Build

    #Webpack Build
    $ npm run build
